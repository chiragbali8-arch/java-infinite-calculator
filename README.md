# java-infinite-calculator
A simple console based infinite calculator built in Java using Scanner, loops, and switch-case statements.

A simple console-based calculator built using Java. This calculator performs continuous calculations until the user press = to exit.

Features 

Addition(+)

Subtraction(-)

Multiplicatiom(*)

Division(/)

Modules(%)

Continuous Calculations

Exit using =

Technologies used

Java 

Scanner class

Loops

Switch case

How to run

1 Compile the program

javac Calculator.java

2 Run the program

java Cakculator
